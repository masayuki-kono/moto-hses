# Add Agent Skills

## Overview

This command adds the required agent skills to the project from the `masayuki-kono/agent-skills` repository.

## Command

```bash
npx add-skill masayuki-kono/agent-skills -s hses-protocol moto-hses-usage hses-packet-analysis rust-code-quality-guide -a cursor -y
```

## Skills Included

- **hses-protocol**: HSES (High Speed Ethernet Server) protocol specification for Yaskawa robot controllers
- **moto-hses-usage**: moto-hses Rust crate usage guide
- **rust-code-quality-guide**: Code quality guide for Rust
